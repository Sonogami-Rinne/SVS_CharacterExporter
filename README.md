# Summer Vacation Scramble Character Exporter
A BepInEx plugin for Summer Vacation Scramble to export characters to PMX.

## How to use

Put the .dll to plugin folder

---

Press Ctrl + Tab in the character maker, then a menu will appear

<img width="590" height="722" alt="屏幕截图 2026-04-20 180012" src="https://github.com/user-attachments/assets/2a4450a3-94fa-4405-a697-7ce6dba36873" />


---

Click the Export button, a folder will pop up after a few minutes. (This takes longer than the time required by the KK_Exporter)

<img width="812" height="662" alt="屏幕截图 2025-09-21 164523" src="https://github.com/user-attachments/assets/4a453863-96f9-4916-a6bc-dbe2491768c2" />

---

Then Install the plugin for blender(modified from ([KKBP](https://gitgoon.dev/kkbp-dev/KKBP_Importer)) to import the model.pmx in the folder (not the one in outfit folder)

<img width="527" height="571" alt="屏幕截图 2025-09-21 164546" src="https://github.com/user-attachments/assets/71ab8fdd-6c01-48e0-b357-adaffba1df3a" />

---

After minutes, the model will appear

<img width="608" height="748" alt="屏幕截图 2025-10-06 172804" src="https://github.com/user-attachments/assets/21299e41-df49-447d-b510-9bccf3fa909c" />


---
## Other Export Options

### ExportAllOutfits
One click for exporting all outfit variations of current character

### ExportWithMainCamera
An experimental function.This will take more time but export better textures.
